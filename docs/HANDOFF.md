## Phase 3 Spec (auth screens)

All screens live in `app/(auth)/` — the layout is already a centred single-column wrapper.

### Welcome `/welcome`
Logo mark (gradient circle "P"), "Partii" gradient wordmark, tagline "Where the party plans itself" (text-secondary), Get Started → /signup (primary orange), Sign In → /login (secondary lavender), OAuth row (Google, GitHub), small disclaimer at bottom.

### Login `/login` *(verify existing file)*
- `react-hook-form` + `zod`: email, password
- `authApi.login` → POST `/api/auth/set-tokens` → `router.push('/dashboard')`
- Forgot password link, OAuth row, "Sign up" footer link

### Sign Up `/signup`
- Fields: displayName, email, password (show/hide toggle), DOB
- 18+ client-side validation — reject if under 18
- `authApi.signup` → POST `/api/auth/set-tokens` → `router.push('/verify-email?email=...')`
- OAuth row, "Sign in" footer link

### Email Verification `/verify-email`
- Uses `<OtpInput>` component (6 boxes, auto-advance, paste)
- Read `?email=` from searchParams for subtitle
- `authApi.verifyEmail({ email, otp })` → redirect `/dashboard`
- 60s resend countdown → `authApi.resendOtp(email)` when expired

### Forgot Password `/forgot-password`
- Email input → `authApi.forgotPassword({ email })`
- On success: replace form with success state (lavender checkmark circle + message)

### Reset Password `/reset-password`
- Read `?token=` from searchParams
- New password + confirm password (must match, min 8 chars)
- `authApi.resetPassword({ token, newPassword })` → success state → redirect `/login` after 2s

### OAuth Callback `/oauth-callback`
- `'use client'`, reads `?access_token=&refresh_token=` from URL on mount
- POST to `/api/auth/set-tokens` → redirect `/dashboard`
- Loading spinner while processing; error state if tokens missing

### OtpInput component `components/ui/OtpInput.tsx`
- 6 `<input maxLength={1} type="text" inputMode="numeric">` boxes
- Styling: `w-11 h-14 text-center bg-bg-surface border border-border rounded-input focus:border-accent-primary focus:shadow-lavender-ring outline-none text-h1 text-text-primary`
- Auto-advance on digit entry, backspace goes to previous box
- Paste splits across all 6 boxes
- Props: `value: string`, `onChange: (val: string) => void`

## Key Architecture Facts

- **Token storage:** httpOnly cookies (`partii_access` 15m, `partii_refresh` 7d)
- **Client can't read tokens** — client forms POST to `/api/auth/set-tokens` after login
- **Server components** call `getAccessToken()` from `lib/auth/tokens.ts`
- **Auth guard:** `middleware.ts` reads cookie, redirects unauthenticated → `/login?redirect=...`
- **Backend URL:** `NEXT_PUBLIC_API_URL=http://localhost:8080/partii/api/v1` (in `.env.local`)
- **API client:** `lib/api/client.ts` → throws `ApiClientError` on non-2xx

## Design System Rules (enforce on every component)
- Dark only — `bg-bg-primary` (#000) base
- **Orange** = actions (buttons, CTAs). **Lavender** = navigation, info, selected states
- Phosphor icons thin weight; `@phosphor-icons/react`
- Focus ring: `focus:border-accent-primary focus:shadow-lavender-ring`
- Gradient text: `style={{ background: 'linear-gradient(135deg, #9B8EC4, #FF8C42)', WebkitBackgroundClip: 'text', WebkitTextFillColor: 'transparent' }}`
- Use `VelvetButton` (not shadcn `button.tsx`) — it has `loading`, `variant`, `size` props
